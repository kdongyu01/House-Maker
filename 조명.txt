주택 내 각 부분별 전등 구매

1. 총 필요한 전등 개수 계산 

- 방 4개당 1개 / 화장실 2개에 각 1개 / 주방 2개 / 거실 3등짜리 1개 / 현관 1개 / 다용도실 1개 / 창고 1개 / 주차장 1개 

총 약 15개 (추가분)

2. 각 방 별 전등 탐색

- 사각형 LED 방전등
- 원형 LED 방전등
- 형광등
- 할로갠 햄프
- 메탈 힐라이드 램프
- 수은등 

3. 전등 별 소요 전력량 측정

- 사각형 LED 방전등: 약 50~200W
- 원형 LED 방전등: 약 30~150W
- 형광등: 약 1040W (일반), 40110W (고출력)
- 할로겐 램프: 약 20~1000W
- 메탈 핼라이드 램프: 약 70~2000W
- 수은등: 약 50~1000W

4. 전등 설치 업체 탐색

원하 LED / 포커스 조명 / 세명엘텍LED 조명 / 디엘라이팅 / 엘이디시대 등등

5. 전등 구매 가격 계산

LED 방등 약 20만원 기준 15개 -> 300만원
