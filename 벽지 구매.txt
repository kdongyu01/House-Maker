주택 내 방 4개 및 거실 주방 등에 바를 벽지

1. 벽지의 대략적인 가격대 측정

실크 벽지 : 1롤 당 7450원 -> 약 75000원
뮤럴 벽지 : 1롤당 약 19000원 -> 190000원
합지 벽지 : 1롤 당 2900원 -> 약 30000원

2. 벽지가 필요한 면적 계산

- 벽지 면적 : 5평당 1롤 (1 Roll) -> 50평 : 10롤 


3. 벽지 종류 탐색

- 벽지 종류 : 실크 벽지, 합지 벽지, 뮤렬 벽지


4. 각 벽지의 장단점 비교

- 합지 벽지
장점 : 가격이 저렴함, 친환경적임, 통퓽성이 좋음, 인체에 무해 
단점 : 변색, 오염, 습기에 약한 내구성

- 실크 벽지
장점 : 내구성, 복원력이 좋음, 장기간 사용하기 좋음, 디자인과 질감 다양성 높음
단점 : 가격이 비쌈, 통풍성이 좋지 않음

- 뮤럴 벽지 
장점 : 부직포 위에 붙여 만들어 벽화를 보는 느낌을 가질 수 있음
단점 : 무늬가 많아 산만해 보일 수 있음
